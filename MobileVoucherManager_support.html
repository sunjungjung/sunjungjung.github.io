꼭 쓰자 - 쿠폰(상품) 매니저

쿠폰(기프티콘, 기프티쇼 등) 여러가지 모바일 쿠폰을 달력 형태로 정리하여 만료일 전까지 잊어버리지 않고 쓸 수 있도록 정리를 도와줍니다.
관리가 필요한 쿠폰을 저장하여 관리해주세요.

상품 추가
상품 추가 페이지에서 사용할 쿠폰 사진을 불러오고, 상품 사용처, 상품 명, 사용 만료일, 메모를 입력하고 저장해주세요
메모는 잔액 확인 가능한 링크가 있는 경우 입력하시면 활용하실 수 있습니다.

달력
달력 형태로 날짜 별 사용 가능한 상품을 확인 가능합니다.
날짜 밑의 동그라미는 사용 가능한 쿠폰이 있는 경우 표시됩니다.
날짜를 선택 하면 하단에 사용 가능한 쿠폰 목록을 확인할 수 있습니다.
하단의 목록을 선택하면 해당 항목의 자세한 사항을 확인할 수 있습니다.

전체 쿠폰 목록
전체 쿠폰 목록 메뉴에서 현재 기기에 등록된 모든 상품을 표 형태로 확인 가능합니다.
사용처, 상품명, 만료일을 확인할 수 있고, 클릭하면 해당 항목에 대해 자세한 사항이 확인 가능합니다.

상세 내역 확인
사진, 사용처, 상품명, 만료일, 메모를 확인할 수 있습니다.
사용 완료된 경우, 하단의 사용완료 버튼을 눌러 사용완료로 표기하거나, 오른쪽 상단의 휴지통 버튼을 클릭하여 삭제할 수 있습니다.
수정할 사항이 있는 경우, edit 버튼을 눌러 수정이 가능합니다.
